微信文章排版工具

左侧使用 Markdown 编写：# / ## / ### 标题、空行分段、**粗体**、*斜体*、[文字](链接)、插图 ![说明](图片地址)（独占一行为大图）、代码块 ``` … ```。若填写下方 Hello 图床 Token，「插入图片」会上传到图床并插入 https 链接（适合复制到公众号）；未配置时仍为本地 base64。上传过程中页面顶部会出现彩色状态条显示连接 / 上传 / 成功 / 失败，按钮也会变成「上传中…」并禁用；约 2 分钟无响应自动超时。点击复制会将带内联样式的 HTML 写入剪贴板，在公众号后台编辑器中粘贴即可。

Hello 图床 API Token（可选，仅保存在本浏览器）

使用 helloimg.com 开放接口上传；Token 只存放在本机 localStorage，泄漏请立刻在图床重置。请勿把 Token 写进网页源码或 Git。

若粘贴后样式丢失，可在浏览器中允许剪贴板权限；纯文本备份会一并复制。
